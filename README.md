# liri-node-app
Week 10 Homework - Node.js

# What the project does

This project is an activity in using several different node packages. The packages in use are
as follows:

1. "dotenv": "^6.0.0",
2. "file-system": "^2.2.2",
3. "inquirer": "^6.0.0",
4. "node-spotify-api": "^1.0.7",
5. "request": "^2.87.0",
6. "spotify": "^0.3.0",
7. "twitter": "^1.7.1"

# Why the project is useful?

This project will be useful to anyone getting started using node and node packages. It will also be useful to anyone wanting to learn how each package included in the project is used.

# How users can get started with the project?

Just install node on your local machine and clone the repository.
